# PRM - Personal Resource Manager

## 项目概述

个人资源管理App，帮助用户管理人脉、圈子、人情关系。
核心理念：人情世故 + 圈层管理 + 智能提醒。

## 技术架构

- **数据底座**: 钉钉多维表格（联系人基础信息、圈子、互动记录）
- **隐私层**: 本地PWA + localStorage/IndexedDB（人情账本、关系策略、私密备注）
- **展示层**: H5页面（可分享到微信/抖音的联系人卡片、圈子展示）

## 核心概念

| 概念 | 说明 |
|------|------|
| 圈子 | 戈友、沙友、校友、同好、体育等 |
| 人情 | 有恩于我 / 有恩于他 |
| 策略 | 加密(strengthen) / 保持(maintain) / 淡出(phase out) |
| 隐私 | 人情账本和策略标记永远不上云 |
